Data Preparation With Pandas¶

Pandas is a Python library that provides tools to work with tabular and series data. Series data are 1-D data vectors with label. Dataframes are 2-D tabular structures with labels. There exist also a 3-D structures similarly that are called Panles. In this blog, we are focusing on dataframe structures.

Importing Pandas¶

After installing the module, simply import Pandas using import command. It is a convention to import Pandas as pd:

In [1]:
import pandas as pd

Load and prepare the data¶

A critical step in any data analysis task is to prepare the data correctly. Here we load bike sharing history data that is available at UCI Machine Learning Database. You may download the data from here and extract it in the current folder.

In [4]:
data_path = './hour.csv'

rides = pd.read_csv(data_path)

rides is a Pandas dataframe.

Accessing the data¶

Dataframe object is not callable and we cannot use indexing to directly access an entry in the table. Instead, we can use iloc method for slice indexing. It returns a dataframe object that is a slice of the original table:

In [82]:
rides.iloc[0:2, 10:13]
Out[82]:
temp atemp hum
0 0.24 0.2879 0.81
1 0.22 0.2727 0.80

Similarly, we can use loc method but instead of giving indices for slicing, we pass the columns'/rows' labels:

In [83]:
rides.loc[[0,1,2], ['temp', 'atemp', 'hum']]

Plotting the data¶

This dataset we are looking at has the number of riders for each hour of each day from January 1 2011 to December 31 2012. The number of riders is split between casual and registered, summed up in the cnt column. You can see the first few rows of the data above.

To plot the cnt values over time we can call the plot method. x and y and title attributes can be passed to the method. Here, we also switched on the grid:

In [110]:
plt_10_days = rides[:10*24].plot(x='dteday', y='cnt', \
                                 title='Total Count Of Bike Rental', \
                                 grid=True)

Above is a plot showing the number of bike riders over the first 10 days or so in the data set. (Some days don't have exactly 24 entries in the data set, so it's not exactly 10 days.) We can see the hourly rentals here. This data is pretty complicated! The weekends have lower over all ridership and there are spikes when people are biking to and from work during the week. Looking at the data above, we also have information about temperature, humidity, and windspeed, all of these likely affecting the number of riders.

In [88]:
plt_bar = rides.iloc[0:24,14:16].plot.bar(stacked=True,figsize=(10,3))

The cumulative summation of additive quantities is an intersting feature that can be readily obtained:

In [63]:
plt_cumsum = rides.iloc[:10*24, 14:17].cumsum().plot(figsize=(10,3))

Finally, with stochastic data it is interested to see the moving average of data in a specific time window. We compare two time windows here, the 1-day length and 30-day length for temperature values (normalised by deviding to 41°C):

Dummy variables¶

Here we have some categorical variables like season, weather, month. To include these in modeling, we need to make binary dummy variables. This is simple to do with Pandas thanks to get_dummies(). We can use the drop method to remove those columns (nd others we don't need) from the dataframe:

In [90]:
dummy_fields = ['season', 'weathersit', 'mnth', 'hr', 'weekday']
for each in dummy_fields:
    dummies = pd.get_dummies(rides[each], prefix=each, drop_first=False)
    rides = pd.concat([rides, dummies], axis=1)

fields_to_drop = ['instant', 'dteday', 'season', 'weathersit', 
                  'weekday', 'atemp', 'mnth', 'workingday', 'hr']#,
data = rides.drop(fields_to_drop, axis=1)

Scaling target variables¶

It is normally needed to make the variables in the same range, when performing a similar analysis to data with various scales. That is, we'll shift and scale the variables such that they have zero mean and a standard deviation of 1.

In [92]:
quant_features = ['casual', 'registered', 'cnt', 'temp', 'hum', 'windspeed']

scaled_features = {}

for each in quant_features:
    mean, std = data[each].mean(), data[each].std()
    scaled_features[each] = [mean, std]
    data.loc[:, each] = (data[each] - mean)/std

Finally, we plot the quantity features before and after the normalization for comparison.

In [93]:
plt_raw = rides.loc[:24, quant_features].plot(figsize=(10,3),\
                                              title='Various Scale Quantity Series')
plt_nrm = data.loc[:24, quant_features].plot(figsize=(10,3),\
                                              title='Normalized Quantity Series')
